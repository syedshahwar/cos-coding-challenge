`The project was developed just to resolve the CarOnSale Challenge`

# HOW TO SETUP AND RUN PROJECT
# SETUP

- Install node.js
- Run the command: 
```
npm run start
```

# HOW TO RUN?

Run the command:
```
npm start
```

# 3. HOW TO TEST?

Run the command:
```
npm test
```

# 4. HOW TO SEE THE COVERAGE?

You can run the following commands:
```
npm run coverage
```
